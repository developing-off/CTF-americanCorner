1. What is SQL injection?

A. A type of programming language

B. A security vulnerability in web applications

C. A database management system

D. A data encryption technique

Solution: B. A security vulnerability in web applications

2. How does SQL injection work?

A. By injecting JavaScript code into a website

B. By manipulating the structure of SQL queries through user inputs

C. By encrypting sensitive data in a database

D. By exploiting browser vulnerabilities

Solution: B. By manipulating the structure of SQL queries through user inputs

3. How can SQL injection vulnerabilities be prevented?

A. Implementing input validation and parameterized queries

B. Using secure passwords

C. Disabling database logging

D. Increasing the database timeout

Solution: A. Implementing input validation and parameterized queries

4. Which SQL injection payload is commonly used to bypass authentication?

A. '; DROP TABLE users;--

B. 1=1; --

C. ' OR '1'='1'--

D. SELECT * FROM users

Solution: C. ' OR '1'='1'--

5. In the fixed code example, what is the purpose of the question marks (?) in the SQL query?

A. To comment out parts of the query

B. To indicate a placeholder for user input

C. To concatenate strings in the query

D. To execute a subquery

Solution: B. To indicate a placeholder for user input

6. What additional measure, besides using parameterized queries, helps prevent SQL injection?

A. Least Privilege Principle

B. Using dynamic SQL queries

C. Storing passwords in plain text

D. Avoiding input validation

Solution: A. Least Privilege Principle

7. Which of the following is not a recommended practice for preventing SQL injection?

A. Input validation

B. Using stored procedures

C. Concatenating user input directly into SQL queries

D. Implementing proper authentication and authorization

Solution: C. Concatenating user input directly into SQL queries

8. What is the primary goal of a SQL injection workshop?

A. To teach participants how to write SQL queries

B. To demonstrate the impact of SQL injection attacks

C. To explore the benefits of database normalization

D. To practice secure coding techniques

Solution: B. To demonstrate the impact of SQL injection attacks
